#LTDouBanFM
###简介
自己写的豆瓣FM客户端，swift语言，数据来自百度豆瓣FM，播放页面照着网易云音乐做的

###第三方库
- Alamofire
- SwiftyJSON
- AFSoundManager
- RESideMenu
